# cd-dvd-ripping
small project for ripping CDs/DVDs to ISOs using multiple drives on one machine.
# AI used in this project:
- AI was used as inline suggestions (VS Code built in copilot)
- AI was also used as information gathering such as finding the right tool for DVD metadata.
- Other use of AI will be marked in relevent locations.
# Requirements:
The system is tested on debian only.
## Required Packages
- `lsdvd`
- `libcdio-utils`
- `pyyaml`
